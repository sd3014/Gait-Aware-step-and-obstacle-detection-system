# Gait-Aware-step-and-obstacle-detection-system
This Adaptive Walking Assistant uses a smartphone camera and ML (MiDaS, YOLOv8, LSTM) to provide personalized, real-time guidance to visually impaired users. It detects step heights and obstacles, and assesses gait stability using SisFall-trained models. Guidance is adaptive via voice or AR, promoting safe, independent mobility.
